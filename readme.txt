-----------------------------------------------------
■启动方法
-----------------------------------------------------
双击[东方梦黎明.jar]或压缩包中[东方梦黎明.exe] 启动游戏。

-----------------------------------------------------
■游戏概要
-----------------------------------------------------
标题：        东方梦黎明
类型：        恐怖解谜冒险类
预测游戏时间：1分钟 ― 半小时
游玩对象年龄：12岁以上
结局数量：    2

※游戏中有惊吓、流血等画面，请胆小的玩家不要尝试。

-----------------------------------------------------
■操作方法
-----------------------------------------------------
主人公的移动：方向键（↑／←／→／↓）
确定：         Z键
取消菜单：     X键
跑步：         Z键
回到标题画面： C键调出存档界面，选择回到标题画面
存档：         C键存档，X键退出存档界面
读档：         在菜单画面中读档
游戏结束：     关闭窗口或开始界面选择结束游戏

-----------------------------------------------------
■物品使用方法
-----------------------------------------------------
物品自动使用。
得到物品后，移动到达对应的目的地，触发事件。

-----------------------------------------------------
■关于攻略
-----------------------------------------------------
详见课程设计总结报告。

-----------------------------------------------------
■运作环境
-----------------------------------------------------
OS Microsoft(R) Windows(R) 2000 / XP / Vista / 7 / 8 / 10 中文版
CPU Intel(R) Pentium(R) III 1.0GHz及以上
内存 256MB以上
显示器 分辨率800×600以上

※没有安装Java虚拟机环境的用户请使用EXE版本，已安装的用户可以选择JAR版本。
